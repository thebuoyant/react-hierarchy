# React 19 + TypeScript + MUI + Zustand + Vite + Vitest + ESLint + Storybook

Dieses Repo ist ein **lauffähiges Setup** für Komponenten-Entwicklung in Storybook (React/Vite) plus **Mini-Webapp** zum Verifizieren im Browser.

## Voraussetzungen
- Node.js (aktuelle LTS empfohlen)
- npm

## Installation
```bash
npm install
```

## Mini-Webapp starten
```bash
npm run dev
```
Danach im Browser öffnen (Vite zeigt die URL an).

## Storybook starten
```bash
npm run storybook
```

## Tests
```bash
npm test
```

## Lint
```bash
npm run lint
```

## Was ist drin?
- Beispiel-Komponente: `CounterCard` (MUI Card + Buttons)
- Globaler Zustand über Zustand: `counterStore`
- Storybook Stories + Interactions-ready Addons
- Vitest + Testing Library + jest-dom Setup
